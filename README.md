# LeagueOfLegends
## Работа с json файлом
### Разработан на основе:
- React
- Redux
- Antd
- Typescript
- Swiper
#### Описание
- Работа с json файлом LeagueOfLegends
- Список всех персонажей лол
- Каждый персонаж имеет личную информацию
- На личной информации персонажа есть карусель его всех скинов, а также есть калькулятор показания силы в зависисмоти от уровня
